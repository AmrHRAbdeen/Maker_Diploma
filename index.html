Day Summary"CAD and 3D printing"

3D printing: Fused Deposition Modelling (FDM)

STL : Standard Tesselation Language used to save designs for 3D printing (extension).

Slicing(Chop Shop): From 3D to 2D ---> Creating the toolpath.

3D printer components [FDM]:

  1. The Jet [ Hot End ]
  2. Feeder
  3. Motion Stage : Delta and Cartesian
  4. Build platform

You can print not only plastic but also metal!BOEING uses this technology for jet engines :O

Materials used in 3D printing:

  1. PLA [ PolylacticAcid ]
    • Easy to print
    • safe to print
    • usen when strength is not critical.
    • Temp. used for it :170 -> 230 °C,, Bed : 30 -70 °C
    • In Egypt : 1 Kilo of pla 750 LE ( 2017 )

  2. ABS [ Acrylonitrile butadiene styrene ]
    • Hard to print
    • needs a heated platform
    • High strength application
    • High temp. Applications
    • Temp. used for it :230 -> 270\280 ,, Bed: 110-130
    • In market: LEGO toys.

  3. PVA [polyvinyl alcohol]
    • Frequently used with ABS

  4. Nylon

Terminologies for 3D printing:

  1. Infill :Ratio of Material to Air EX. if Infill is 10 % then Material is 10 % and Air is 90 % and so on...
  2. OverHang test
  3. Bridging test
  4. Layer Resolution = Layer Height.
  5. Teardrop
  6. Raft: By default is 1 to 1.5 mm to maintain a smooth surface for the bottom of the design.
  • 3D printer:[In Fusion 360] *select Bodies then Save as STL [ ASCII ].
  • Laser Cutter: [In Fusion 360] *Select Sketch then Save as DXF.
